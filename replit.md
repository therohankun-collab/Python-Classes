# Python Learning Project

## Overview
A beginner-level Python learning project organized into lessons and activities. It covers fundamental Python concepts such as printing, variables, keywords, and conditional logic.

## Project Structure

```
Lesson 1 Activity 1/
  main.py                          - Basic print statements demo
  Keywords.py                      - Lists Python reserved keywords
  Understanding_Variables_In_Python.py  - Variable assignment and user input
  Lesson_2_Afterclass_Project.py   - Birthday lookup tool using if-elif-else
```

## Language & Runtime
- **Language:** Python 3.12
- **Environment:** Replit Nix (stable-25_05 channel)
- **Package Manager:** UPM with python3-uv backend

## Workflow
- **Start application** — runs `python3 'Lesson 1 Activity 1/main.py'` as a console workflow
